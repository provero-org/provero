# Support

## How to get help

- **Questions and ideas**: [GitHub Discussions](https://github.com/provero-org/provero/discussions)
- **Bug reports**: [GitHub Issues](https://github.com/provero-org/provero/issues)
- **Real-time chat**: [Slack](https://provero.slack.com)
- **Security issues**: see [SECURITY.md](SECURITY.md)

## Before opening an issue

1. Check existing [issues](https://github.com/provero-org/provero/issues) and [discussions](https://github.com/provero-org/provero/discussions)
2. Include your Provero version (`provero version`)
3. Include your `provero.yaml` configuration (redact sensitive values)
4. Include the full error output
